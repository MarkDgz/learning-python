

English
=======
This program insert custom watermark on images located at input directory and generating all of them on output directory.

Procedure:

1.Place the images on inp-img directory
2.Modify the parameters with "nano wm.py", save it.
3.Run the program with this command: python3 wm.py
4.Collect the result images on out-img directory

Spanish
=======
Este programa inserta una marca de agua personalizada en las imágenes ubicadas en el directorio de entrada y las genera todas en el directorio de salida.

Procedimiento:

1.Coloque las imágenes en el directorio inp-img
2.Modifique los parámetros con "nano wm.py", guárdelo.
3.Ejecute el programa con este comando: python3 wm.py
4.Recopile las imágenes de resultados en el directorio out-img
